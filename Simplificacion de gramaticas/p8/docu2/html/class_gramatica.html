◆ encontrandoParejas()

void Gramatica::encontrandoParejas ( )

Método de la ALGORITMO DE PRODUCCIONES UNITARIAS: Se encarga de, primero cojo la produccioe entera y busco si en la parte derecha hay un solo no terminal en la produccion. Si se cumple, inserto en el set de parejas donde el primero que inserto es la parte izq y el segundo es el no terminal. Luego recorro el set de parejas y busco si tengo guardada alguna pareja en la que el segundo sea igual al primero de otra pareja. Por último, intercambio en las parejas: En la parte izquierda inserto las producciones de la parte derecha
